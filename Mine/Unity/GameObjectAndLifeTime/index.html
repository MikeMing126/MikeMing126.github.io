  • Awake еҮәз”ҹж—¶и°ғз”Ё зұ»дјјдәҺжһ„йҖ еҮҪж•°гҖӮ

    Unity calls Awake when an enabled script instance is being loaded.

    This function is called when the object becomes enabled and active.

  • Start з¬¬дёҖж¬ЎUpdateеүҚи°ғз”ЁпјҢд»…дёҖж¬Ў(еҰӮжһңеңЁAwakeдёӯзӣҙжҺҘSetActiveзҰҒз”ЁпјҢеҲҷдёҚдјҡи°ғз”ЁгҖӮ)

    Start is called on the frame when a script is enabled just before any of the Update methods are called the first time.

  • FixedUpdate зү©зҗҶеё§жӣҙж–° еӣәе®ҡй—ҙйҡ”(Timestep)жү§иЎҢгҖӮ

    еңЁжёёжҲҸеҚЎйЎҝж—¶пјҢжү§иЎҢж¬Ўж•°дјҡеўһеҠ гҖӮеӣ дёәжҳҜжҢүеҜ№еә”зҡ„еӣәе®ҡй—ҙйҡ”еҺ»жү§иЎҢзҡ„гҖӮ
    еҸҜд»ҘеңЁProject Settings->Time->Fixed TimestepеҺ»дҝ®ж”№гҖӮ

    This function is called when the behaviour becomes disabled.
