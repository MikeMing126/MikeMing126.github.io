Unity UI(UGUI) is a GameObject-based UI system that you can use to develop user interfaces for games and applications.

еҹәжң¬зҡ„UGUIз•Ңйқўпјҡ

  • Canvas з”»еёғгҖӮжүҖжңүз»ҳеҲ¶зҡ„UIе…ғзҙ еә”иҜҘзҪ®дәҺз”»еёғд№ӢдёӢгҖӮ

UnityдёӯжёІжҹ“зҡ„зү©дҪ“йғҪжҳҜз”ұMeshжһ„жҲҗзҡ„гҖӮйҮҚжһ„еҚіCanvasиҙҹиҙЈе°ҶеӯҗиҠӮзӮ№зҡ„UIе…ғзҙ зҡ„зҪ‘ж јеҗҲе№¶пјҢе№¶з”ҹжҲҗзӣёеә”зҡ„жёІжҹ“жҢҮд»ӨеҶҚеҸ‘йҖҒеҲ°Unityзҡ„еӣҫеҪўз®ЎйҒ“зҡ„иҝҮзЁӢгҖӮ

еҗҢдёҖдёӘCanvasдёӢзҡ„д»»ж„Ҹе…ғзҙ еҸҳеҢ–пјҲдҪҚзҪ®гҖҒйўңиүІзӯүгҖҒActiveзҠ¶жҖҒпјүпјҢдјҡи§ҰеҸ‘ж•ҙдёӘCanvasзҡ„йҮҚз»ҳгҖӮ
Rebuild - дё»иҰҒеҲҶдёәеёғеұҖйҮҚжһ„Layout Rebuildе’Ңз»ҳеҲ¶йҮҚжһ„Graphic RebuildгҖӮ
еүҚиҖ…дё»иҰҒжҳҜе°әеҜёгҖҒдҪҚзҪ®гҖҒеёғеұҖи§„еҲҷзӯүпјҢеҗҺиҖ…дё»иҰҒжҳҜйўңиүІгҖҒй—®йўҳзӯүеӣҫеҪўеұһжҖ§гҖӮ
Canvasзҡ„BatchеҸӘдјҡеҪұе“Қе…¶еӯҗиҠӮзӮ№пјҢе№¶дёҚдјҡеҪұе“ҚеӯҗCanvasгҖӮ

дјҳеҢ–зӯ–з•Ҙ

  • йў‘з№ҒеҸҳеҢ–зҡ„UIдёҺеҹәжң¬дёҚеҸҳзҡ„еҲҶCanvasеӨ„зҗҶпјҢеҮҸе°‘еҸҳеҢ–иҖ—ж—¶
  • дҪҝз”Ёеқҗж Үз§»еҠЁж–№ејҸжқҘд»ЈжӣҝSetActiveжҳҫйҡҗ

Sprite Atlas

An asset that consolidates several textures into a single combined texture. Unity can call this single texture to issue a single draw call instead of multiple draw calls to access the packed textures all at once at a smaller performance overhead.

йңҖиҰҒе®үиЈ…еҜ№еә”зҡ„жҸ’д»¶гҖӮPacakgeManager -> 2D Sprite
еҗҜз”ЁеңЁProjectSettings - Editor - SpritePackerйҮҢи®ҫзҪ®ModeгҖӮ

еұһжҖ§

Type - еӣҫйӣҶзұ»еһӢгҖӮдё»еӣҫйӣҶ/еҸҳдҪ“еӣҫйӣҶгҖӮеҸҳдҪ“еӣҫйӣҶз”ЁдәҺеҲӣе»әе’Ңдё»еӣҫйӣҶдёҚеҗҢеҲҶиҫЁзҺҮзҡ„еӣҫйӣҶгҖӮ
Include in Build - жү“еҢ…йЎ№зӣ®иҮӘеҠЁеҢ…еҗ«еӣҫйӣҶпјҢе№¶дё”еңЁиҝҗиЎҢдёӯиҮӘеҠЁеҠ иҪҪгҖӮеҰӮжһңдёҚиҮӘеҠЁеҠ иҪҪпјҢеҲҷйңҖиҰҒиҝӣиЎҢеҗҺжңҹз»‘е®ҡгҖӮ
Allow Rotation - дёҖиҲ¬еҗҜз”ЁпјҢжңҖеӨ§йҷҗеәҰеҲ©з”Ёз©әй—ҙпјҢдҪҶеҜ№дәҺCanvas UIе…ғзҙ зҡ„зә№зҗҶпјҢйңҖиҰҒзҰҒз”ЁгҖӮ
Padding - еЎ«е……пјҢйҒҝе…ҚзӣёйӮ»зҡ„Spriteд№Ӣй—ҙеҮәзҺ°PixelйҮҚеҸ гҖӮ
Read/Write Enabled - еҗҜз”ЁеҗҺдјҡзҝ»еҖҚеҶ…еӯҳпјҢдјҡеҲӣе»әеүҜжң¬гҖӮд»…йҖӮз”ЁдәҺжңӘеҺӢзј©жҲ–DXTеҺӢзј©зә№зҗҶпјҢеҗҜз”ЁеҗҺеҸҜд»ҘйҖҡиҝҮTexture2D.SetPixelsзӯүи„ҡжң¬и®ҝй—®зә№зҗҶж•°жҚ®гҖӮ

жңүдёӨзҜҮеҸӮиҖғж–Үз« пјҡ
гҖҗUnityжёёжҲҸејҖеҸ‘гҖ‘SpriteAtlasдёҺAssetBundleжңҖдҪійЈҹз”Ёж–№жЎҲ
еӣҫйӣҶжү“еҢ…дёҺеҠ иҪҪй—®йўҳ

еҰӮжһңдҪҝз”ЁSprite Atlas,еҜ№еә”зҡ„spriteжң¬иә«еңЁжІЎжңүиў«зӣҙжҺҘеј•з”Ёзҡ„жғ…еҶөдёӢеҸҜд»ҘдёҚжү“ABгҖӮ

YooAssetеңЁеӨ„зҗҶABжһ„е»әж—¶пјҢжіЁж„ҸдҪҝз”ЁBuiltinBuildPipelineгҖӮдҪҝз”Ёй»ҳи®Өзҡ„ScriptableBuildPipelineдјҡеӯҳеңЁжһ„йҖ еҶ—дҪҷABзҡ„й—®йўҳгҖӮ

еӣҫйӣҶдјҳеҢ–

1.еӨ§еӣҫдёҚз”ЁеӣҫйӣҶ
2.иҫ№з•Ңжғ…еҶөпјҢеҰӮеӨҡдёҖдёӘеӣҫд»Һ1024x1024еҸҳдёә1024x2048зҡ„жғ…еҶө - еҸҜд»ҘеҲҶеӨҡдёӘеӣҫйӣҶгҖӮ
3.жіЁж„ҸеӣҫйӣҶд№ұз”Ёзҡ„жғ…еҶөпјҢеҢәеҲҶе…¬е…ұеӣҫйӣҶе’ҢеҚ•зӢ¬з•Ңйқўзҡ„еӣҫйӣҶзӯүгҖӮ

Event System

A way of sending events to objects in the application based on input

еңәжҷҜдёӯеҸӘиғҪжңүдёҖдёӘEventSystem.

е…·дҪ“иҖҢиЁҖпјҢеҢ…еҗ«д»ҘдёӢеҠҹиғҪпјҡ

  • з®ЎзҗҶе“ӘдёӘжёёжҲҸеҜ№иұЎиў«и®ӨдёәжҳҜйҖүдёӯзҡ„
  • з®ЎзҗҶжӯЈеңЁдҪҝз”Ёзҡ„иҫ“е…ҘжЁЎеқ—
  • з®ЎзҗҶе°„зәҝжЈҖжөӢ(еҰӮжһңйңҖиҰҒ) пјҲз»“еҗҲCanvasзҡ„GOдёҠзҡ„Graphic RaycasterзӯүгҖӮ3Dзҡ„жЈҖжөӢд№ҹжҳҜз”ұEventSystemжқҘеӨ„зҗҶзҡ„гҖӮпјү
  • ж №жҚ®йңҖиҰҒжӣҙж–°жүҖжңүиҫ“е…ҘжЁЎеқ—

иҫ“е…ҘжЁЎеқ— Input Modules

  • еӨ„зҗҶиҫ“е…Ҙ
  • з®ЎзҗҶдәӢд»¶зҠ¶жҖҒ
  • е°ҶдәӢд»¶еҸ‘йҖҒеҲ°еңәжҷҜеҜ№иұЎгҖӮ

е°„зәҝжҠ•е°„ Raycaster

  • Graphic Raycaster з”ЁдәҺUIе…ғзҙ 
  • Physics 2D Raycaster з”ЁдәҺ2Dзү©зҗҶе…ғзҙ 
  • Physics Raycaster з”ЁдәҺ3Dзү©зҗҶе…ғзҙ 

еә”з”Ёз»ҸйӘҢе’ҢжіЁж„ҸдәӢйЎ№

еҸҜд»ҘйҖҡиҝҮеңЁCanvasдёӢеҲӣе»әеҜ№еә”зҡ„UIе…ғзҙ е№¶жӢ–жӢҪеҲӣе»әеҜ№еә”зҡ„UIйў„еҲ¶дҪ“гҖӮUIйў„еҲ¶дҪ“йңҖиҰҒж”ҫзҪ®еңЁCanvasдёӢжүҚеҸҜд»ҘжӯЈзЎ®еӨ„зҗҶгҖӮ

жҖ§иғҪдјҳеҢ–

еҰӮдҪ•еҝ«йҖҹдјҳеҢ–жүӢжёёжҖ§иғҪй—®йўҳпјҹд»ҺUGUIдјҳеҢ–иҜҙиө·
еҰӮдҪ•дјҳеҢ–UIдёӯеӨ§йҮҸдҪҝз”ЁSetActiveзҡ„й—®йўҳ

  1. еҗҲзҗҶзҡ„еҲҶй…ҚеӣҫйӣҶ
    еҗҢUIз•ҢйқўеҗҢеӣҫйӣҶгҖӮе…ұз”ЁеӣҫйӣҶгҖӮдёҚеҗҢж јејҸзҡ„еӣҫзүҮж”ҫзҪ®дәҺдёҚеҗҢеӣҫйӣҶпјҢ
  2. дјҳеҢ–CPUејҖй”Җ
    CPUй«ҳејҖй”Җдё»иҰҒжҳҜCanvasзҡ„йҮҚе»әгҖӮCanvas.SendWillRenderCanvasesжҲ–Canvas.BuildBatchгҖӮ
    • еҠЁйқҷеҲҶзҰ»гҖӮ
    • еҮҸе°‘UIе…ғзҙ зҡ„Enableе’ҢDisableпјҢжӣҝд»ЈдёәеҜ№canvasRenderгҖӮиҝӣиЎҢж“ҚдҪңгҖӮ
    • ж…Һз”ЁTextзҡ„Best FitгҖӮ
    • ж…Һз”ЁCanvasзҡ„Pixel PerfectгҖӮ
    • ScrollViewдҪҝз”Ёзј“еӯҳжұ гҖӮзј“еӯҳеҜ№иұЎд№ҹдёҚз”ЁenableжҲ–disableпјҢиҖҢжҳҜж”ҫеҲ°зј“еҶІжұ дёӯгҖӮ
    • йқһеҝ…иҰҒзҡ„еӣҫзүҮзҰҒз”Ёraycast

UI еҗҲжү№и§„еҲҷ

UGUI з•ӘеӨ–д№ӢеҗҲжү№

жң¬иҙЁдёҠиҝҳжҳҜжёІжҹ“MeshгҖӮиҝҷдёӘиҝҮзЁӢдёәCanvasзҡ„RebuildгҖӮ

е…·дҪ“иҖҢиЁҖпјҢзңӢеҲ°еҜ№еә”зҡ„RebuildйҖ»иҫ‘гҖӮ

  1. и®Ўз®— Canvas alphaпјҢеҢ…жӢ¬зҲ¶ Canvas е’ҢеөҢеҘ— CanvasпјҲalpha=0дёҚз”ҹжҲҗеҗҲе№¶жү№ж¬ЎпјүгҖӮ
  2. UI еұӮж¬Ўз»“жһ„еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢжӣҙж–° Batch йЎәеәҸпјҢеҜ№ Canvas дёӢжүҖжңүUIе…ғзҙ пјҲCanvasRendererпјүжҢү UI еұӮж¬Ўз»“жһ„ж·ұеәҰдјҳе…ҲжҺ’еәҸпјҢз”ҹжҲҗ UI InstructionsгҖӮ
  3. жӣҙж–°жүҖжңүйңҖиҰҒеҗҢжӯҘж•°жҚ®зҡ„ renderer UI ж•°жҚ®пјҢеҢ…жӢ¬ vertex, color, material, transform, rect, depthпјҲжҢүUIеұӮж¬Ўз»“жһ„ж·ұеәҰдјҳе…ҲжҺ’еәҸзҡ„ж·ұеәҰпјүзӯүгҖӮйқһжҙ»еҠЁпјҲIsActive() == falseпјүдё”дёҚејәеҲ¶жӣҙж–°зҡ„UIе…ғзҙ пјҢе°ҶдёҚеҗҢжӯҘж•°жҚ®гҖӮ
  4. Canvas ж•°жҚ®жӣҙж–°ж—¶пјҲm_CanvasData.isDirtyпјҢеҰӮ UIеұӮж¬Ўз»“жһ„ж”№еҸҳпјҢеҗҢжӯҘе…ій”®ж•°жҚ®пјҢCanvas.AwakeзӯүпјүпјҢи®Ўз®— UI Instructions зҡ„ depth е№¶жҺ’еәҸгҖҒз”ҹжҲҗ BatchгҖӮ

е…ідәҺDepthзҡ„и®Ўз®—

й»ҳи®Өзҡ„еӨ„зҗҶйҖ»иҫ‘жҳҜжҢүз…§Hierarchyзҡ„йЎәеәҸиҝӣиЎҢж·ұеәҰйҒҚеҺҶзҡ„гҖӮ
еҜ№дәҺдёҖдёӘUIе…ғзҙ пјҢиӢҘе…¶дёҚиҝӣиЎҢжёІжҹ“пјҢеҲҷеҜ№дәҺзҡ„ж·ұеәҰдёә-1гҖӮ
иӢҘе…¶жёІжҹ“:

  • еә•дёӢжІЎжңүе…¶д»–жёІжҹ“е…ғзҙ зҡ„пјҢDepth = 0.
  • еә•дёӢжңүе…¶д»–жёІжҹ“е…ғзҙ зҡ„пјҢеҜ№еә•дёӢжүҖжңүе…ғзҙ иҝӣиЎҢеҰӮдёӢи®Ўз®—пјҡ
    • иӢҘеҜ№еә”е…ғзҙ (lowUI)еҸҜд»Ҙе’ҢеҪ“еүҚе…ғзҙ еҗҲжү№пјҲMaterialе’ҢTextrueдёҖиҮҙпјүпјҢDepth = max(curDepth, lowUI.depth)
    • иӢҘеҜ№еә”е…ғзҙ (lowUI)дёҚиғҪе’ҢеҪ“еүҚе…ғзҙ еҗҲжү№пјҢDepth = max(curDepth, lowUI.depth)гҖӮ
      з®ҖеҚ•иҜҙе°ұжҳҜдјҡжұӮеҪ“еүҚе…ғзҙ зҡ„жңҖеӨ§DepthгҖӮ

еңЁи®Ўз®—е®ҢжҜ•еҗҺпјҢеү”йҷӨDepthдёә-1зҡ„пјҢжҢүз…§Depth>MaterialID>TextrueID>RenderOrder иҝӣиЎҢжҺ’еәҸпјҢз„¶еҗҺе°қиҜ•иҝӣиЎҢеҗҲжү№гҖӮ(RenderOrderдҫқжҚ®HierarchyйЎәеәҸпјҢеҚіsibling indexеҶіе®ҡпјҢдёҚиҝҮеҫ…иҖғжҚ®)
жҺ’еәҸеҗҺпјҢеҶ…йғЁзӣёйӮ»зҡ„е…ғзҙ пјҢеҰӮжһңз¬ҰеҗҲеҗҲжү№жқЎд»¶пјҢеҚіMaterialIDе’ҢTextrueIDзӣёеҗҢпјҢеҲҷеҸҜд»ҘеҗҲжү№гҖӮ

Maskпјҡ

еҸҜд»ҘеңЁUGUIжәҗз ҒзңӢгҖӮMaskжң¬иә«дјҡжңүдёҖдёӘm_MaskMaterialе’Ңm_UnmaskMaterialгҖӮ
е…¶дҫқиө–дәҺGPUзҡ„жЁЎжқҝжөӢиҜ•пјҡStencil TestгҖӮ

еҜ№дәҺMaskдјҡйўқеӨ–и°ғз”ЁдёӨж¬ЎDrawCallпјҢз¬¬дёҖж¬Ўз”ЁдәҺз»ҳеҲ¶Stencil MaskпјҢз”ЁдәҺжЁЎжқҝжөӢиҜ•пјҢз¬¬дәҢж¬ЎеҲҷз”ЁдәҺжәўеҮәStencil MaskгҖӮMaskеҶ…зҡ„е…ғзҙ еңЁж»Ўи¶іжқЎд»¶жғ…еҶөдёӢеҸҜд»ҘеҗҲжү№пјҢMaskеӨ–зҡ„е’ҢMaskеҶ…зҡ„е…ғзҙ еҲҷж— жі•еҗҲжү№гҖӮ
дёҫдҫӢпјҡ

  • и°ғз”Ёз¬¬дёҖдёӘDrawCallз»ҳеҲ¶Mask з§»йҷӨ (Batch Breaking Reason - Diff Material Instance)
  • и°ғз”Ёз¬¬дәҢдёӘDrawCallз»ҳеҲ¶MaskеҶ…зҡ„е…ғзҙ  (Batch Breaking Reason - Diff Material Instance)
  • и°ғз”Ёз¬¬дёүдёӘDrawCallе°ҶStencil Maskз§»йҷӨ (Batch Breaking Reason - Diff Material Instance)

еӨҡдёӘMaskд№Ӣй—ҙпјҢеҶ…йғЁе…ғзҙ еҸҜд»ҘиҝӣиЎҢеҗҲжү№гҖӮдёҚиҝҮд»Қз„¶йңҖиҰҒжіЁж„ҸеҜ№еә”зҡ„е…ғзҙ дёҚиғҪжңүзӣёдә’йҒ®жҢЎе…ізі»еҗҰеҲҷдёҚиғҪеҗҲжү№гҖӮ

RectMask

еҸҜд»ҘеңЁUGUIжәҗз ҒзңӢгҖӮиҜҰз»Ҷи§ҒPerformClippingеҮҪж•°гҖӮ
еҺҹзҗҶпјҡи®ҫзҪ®canvasRendererзҡ„clipRect

  • и°ғз”Ёз¬¬дёҖдёӘDrawCallз»ҳеҲ¶Mask
  • и°ғз”Ёз¬¬дәҢдёӘDrawCallз»ҳеҲ¶MaskеҶ…зҡ„е…ғзҙ (Batch Breaking Reason - Different Rect Clipping)

еӨҡдёӘ RectMask2D д№Ӣй—ҙж— жі•еҗҲжү№пјҢеҚіжҜҸдёӘ RectMask2D еңЁз»ҳеҲ¶еҶ…йғЁе…ғзҙ ж—¶йғҪдјҡдә§з”ҹдёҖдёӘйўқеӨ–зҡ„ Draw CallгҖӮиҝҷжҳҜеӣ дёәжҜҸдёӘ RectMask2D жүҖеҜ№еә”зҡ„ ClipRect жҳҜдёҚеҗҢзҡ„пјҢCPU йңҖиҰҒеҚ•зӢ¬и®ҫзҪ®жёІжҹ“зҠ¶жҖҒгҖӮ

Maskе’ҢRectMaskдҪҝз”ЁдәӢйЎ№

зӣёдәӨдјҡеҪұе“ҚеҗҲжү№гҖӮйҒ®зҪ©еҢәеҹҹеӨ–зҡ„е…ғзҙ пјҢиҷҪз„¶иў«иЈҒеүӘд№ҹдјҡеҪұе“ҚзӣёдәӨе’ҢDepthи®Ўз®—пјҢеҪұе“ҚеҗҲжү№гҖӮ

Mask дјҡеўһеҠ иҮіе°‘дёӨдёӘ Draw CallпјҢиҖҢ RectMask2D дјҡеўһеҠ дёҖдёӘ Draw CallпјҢдҪҶдәҢиҖ…зҡ„еҗҲжү№жңәеҲ¶дёҚеҗҢпјҢеӣ жӯӨдјҳеҢ–жҖқи·Ҝд№ҹдёҚеҗҢпјҡ

  • еҪ“еңәжҷҜдёӯзҡ„ Mask еҶ…йғЁе…ғзҙ жқҗиҙЁзә№зҗҶзӯүзӣёеҗҢж—¶пјҢдҪҝз”Ё Mask иҰҒдјҳдәҺ RectMask2DпјҢеӣ дёә Mask еҶ…йғЁе…ғзҙ иғҪеӨҹиҝӣиЎҢеҗҲжү№гҖӮ
  • еҪ“еңәжҷҜдёӯзҡ„ Mask еҶ…йғЁе…ғзҙ жқҗиҙЁзә№зҗҶзӯүдёҚеҗҢж—¶пјҢдҪҝз”Ё RectMask2D иҰҒдјҳдәҺ MaskпјҢеӣ дёә Mask дјҡдә§з”ҹжӣҙеӨҡйўқеӨ–зҡ„ Draw CallгҖӮ

еҰӮжһңжғіиҰҒдёҚжү“ж–ӯеҗҲжү№зҡ„Mask? - иҮӘе®ҡд№үShaderпјҢи®Ўз®—еғҸзҙ зӮ№жҳҜеҗҰеңЁжёІжҹ“иҢғеӣҙеҶ…гҖӮпјҲеҫ…йӘҢиҜҒпјү
