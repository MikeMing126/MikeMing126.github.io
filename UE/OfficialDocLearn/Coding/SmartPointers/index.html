жҷәиғҪжҢҮй’Ҳ SmartPointers

еҶҷеңЁжңҖеүҚйқўпјҡ
UObjectеҸҠе…¶еӯҗзұ»зҡ„GCз®ЎзҗҶе’ҢжҷәиғҪжҢҮй’Ҳз®ЎзҗҶжҳҜдёӨеҘ—зі»з»ҹгҖӮдёҚиҰҒж··з”ЁдәҶгҖӮ
UObjectжңүеҜ№еә”зҡ„TWeakObjectPtrдёҺStrongObjectPtrгҖӮ
йқһUObjectзұ»зҡ„жүҚдҪҝз”ЁжҷәиғҪжҢҮй’ҲгҖӮ

жҷәиғҪжҢҮй’Ҳзҡ„еҹәзЎҖйғЁеҲҶеңЁеҸҚе°„зі»з»ҹйӮЈиҫ№жңүд»Ӣз»ҚгҖӮ

-> еңЁдҪҝз”ЁжҢҮй’ҲжҢҮй’Ҳж—¶ еҸҜд»ҘиҪ»жқҫеҢәеҲҶжӢҘжңүиҖ…е’Ңи§ӮеҜҹиҖ…гҖӮ
-> жіЁж„ҸеҶ…еӯҳејҖй”ҖдјҡжҜ”жҷ®йҖҡжҢҮй’ҲеӨ§гҖӮиҝҷзӮ№е’ҢC++еҶ…йғЁзҡ„жҷәиғҪжҢҮй’Ҳе…¶е®һжҳҜдёҖж ·зҡ„гҖӮ

иҝҷйғЁеҲҶеҸҜд»ҘзңӢеҲ°SharePointer.hж–Үд»¶

дё»иҰҒжҳҜеңЁдёҠеұӮдёҡеҠЎдҪҝз”ЁгҖӮ
дҪҝз”Ёзҡ„ж—¶еҖҷжіЁж„Ҹе°ҪйҮҸз”ЁMakeSharedжқҘеҲӣе»әж–°зҡ„еҜ№иұЎпјҢжҲ–иҖ…зұ»з»§жүҝиҮӘTSharedFromThis.

дҫөе…ҘжҖ§и®ҝй—®еҷЁ
йқһдҫөе…ҘжҖ§ - еҜ№иұЎдёҚзҹҘйҒ“жҳҜеҗҰиў«жҷәиғҪжҢҮй’ҲжүҖжҢҒжңүгҖӮ
жғіиҰҒд»ҘTSharedPtrжҲ–TSharedRefи®ҝй—®еҜ№иұЎпјҢйңҖиҰҒз»§жүҝиҮӘTSharedFromThisгҖӮ
з„¶еҗҺжҸҗдҫӣдёӨдёӘжҺҘеҸЈпјҢдёҖдёӘжҳҜ AsSharedпјҢдёҖдёӘжҳҜSharedThis

ClassA: public TSharedFromThis
ClassB: public ClassA

еҲҷеңЁClassBзҡ„еҜ№иұЎдёҠпјҡ
и°ғз”ЁAsShared()иҝ”еӣһзҡ„жҳҜClassAзҡ„TSharedRef[ClassAзӣҙжҺҘз»§жүҝиҮӘTSharedFromThis]
и°ғз”ЁSharedThis(this)иҝ”еӣһзҡ„е°ұжҳҜClassBзҡ„TSharedRefдәҶ.
з„¶еҗҺеҸҜд»Ҙз”ұRefиҪ¬дёәжҢҮй’ҲгҖӮ

жңүдёҖзӮ№жҳҜйңҖиҰҒжіЁж„Ҹзҡ„пјҡ

Do not call AsShared or SharedThis from constructors, since the shared reference is not initialized at that time and will cause a crash or assert.

Static Cast -> зј–иҜ‘ж—¶ - дҫқиө–ејҖеҸ‘иҖ…зЎ®дҝқзұ»еһӢе…је®№жҖ§
Dynamic Cast -> иҝҗиЎҢж—¶ - йңҖиҰҒеҠЁжҖҒиҝӣиЎҢзұ»еһӢжЈҖжҹҘпјҢиҖҢжҷәиғҪжҢҮй’Ҳиҝҷеқ—жІЎжңүRTTI

е…ідәҺиҝҷеқ—зҡ„иҪ¬жҚўеҸҜд»ҘзңӢеҲ°SharedPointerTesting.inlж–Үд»¶гҖӮ
жҜ”еҰӮTSharedRefеҲ°const&зҡ„иҪ¬жҚў:
const float & MyFloat = FloatRef; [зңӢеҲ°SharedPointerдёӯеҜ№дәҺз¬ҰеҸ·зҡ„йҮҚеҶҷ]
const float & MyFloat2 = FloatRef.Get();

TSharedPtrдёҺTSharedRef

TSharedRefдёҺTSharedPtrзҡ„иҪ¬жҚў:
Ref -> Ptrзҡ„йҡҗејҸиҪ¬жҚў: зӣҙжҺҘPtr = Ref
Ptr -> Ref еҒҡеӨҡдёҖдёӘеҲӨе®ҡ: if(Ptr.IsValid()){Ref = Ptr.ToSharedRef()}

PtrеҲӨе®ҡ:

  1. if(Ptr.IsValid()) //IsValid
  2. if(Ptr) //bool
  3. if(Ptr.Get()!=nullptr) //Ptr
